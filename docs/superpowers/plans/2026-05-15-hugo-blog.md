# Hugo 个人博客实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建一个暗黑终端绿风格的 Hugo 个人博客，以博客为主页，含 neofetch 终端区、文章卡片瀑布流、快速入口，以及聊天室/音乐/个人空间等子页面。

**Architecture:** 从零构建 Hugo 自定义主题 `terminal-blog`，使用原生 CSS（CSS Custom Properties 设计令牌），无外部 CSS 框架。首页三区域（neofetch + 瀑布流 + 入口卡片），文章页含 TOC 和代码块复制功能，子页面通过 iframe/嵌入方式集成。打字机效果用原生 JS 实现。

**Tech Stack:** Hugo (extended)、原生 CSS、原生 JavaScript、JetBrains Mono + Fira Code + Noto Sans SC 字体

## Task 1: 安装 Hugo 并初始化站点

**Files:**
- Create: `hugo.toml`
- Create: `themes/terminal-blog/theme.toml`

- [ ] **Step 1: 安装 Hugo**

Windows 环境，使用 winget 或 scoop 安装 Hugo extended 版本：

```bash
# 方式一：winget
winget install Hugo.Hugo.Extended

# 方式二：scoop
scoop install hugo-extended
```

- [ ] **Step 2: 验证安装**

```bash
hugo version
```

预期输出包含 `hugo extended` 字样，版本号 >= 0.120.0。

- [ ] **Step 3: 初始化 Hugo 站点**

```bash
cd ~/Desktop/个人博客
hugo new site . --force
```

这会生成默认的 `hugo.toml`、`content/`、`themes/` 等目录。

- [ ] **Step 4: 编写站点配置 `hugo.toml`**

```toml
baseURL = "https://your-domain.com"
languageCode = "zh-cn"
defaultContentLanguage = "zh"
title = "我的博客"
theme = "terminal-blog"
paginate = 12

[params]
  author = "你的名字"
  description = "暗黑终端风格的个人博客"
  terminalGreen = "#00ff41"
  showReadingTime = true
  showWordCount = true

[markup]
  [markup.highlight]
    style = "monokai"
    lineNos = true
    codeFences = true
    guessSyntax = true
    noClasses = true
  [markup.goldmark]
    [markup.goldmark.renderer]
      unsafe = true

[menu]
  [[menu.main]]
    name = "首页"
    url = "/"
    weight = 1
  [[menu.main]]
    name = "关于"
    url = "/about/"
    weight = 2
  [[menu.main]]
    name = "音乐"
    url = "/music/"
    weight = 3
  [[menu.main]]
    name = "空间"
    url = "/space/"
    weight = 4
  [[menu.main]]
    name = "聊天室"
    url = "/chat/"
    weight = 5
  [[menu.main]]
    name = "友链"
    url = "/links/"
    weight = 6
```

- [ ] **Step 5: 创建主题骨架**

```bash

```

- [ ] **Step 6: 编写 `themes/terminal-blog/theme.toml`**


- [ ] **Step 7: 验证 Hugo 可运行**

```bash
cd ~/Desktop/个人博客
hugo server -D
```

预期：Hugo 启动本地服务器（默认 http://localhost:1313），页面为空白但无报错。按 Ctrl+C 停止。

- [ ] **Step 8: 初始化 Git 并提交**

```bash
cd ~/Desktop/个人博客
```

```javascript
(function () {
  'use strict';

  function tocInit() {
    const tocNav = document.getElementById('toc-nav');
    const content = document.getElementById('post-content');
    if (!tocNav || !content) return;

    const headings = content.querySelectorAll('h2, h3');
    const links = tocNav.querySelectorAll('a');

    if (!headings.length || !links.length) return;

    // 高亮当前章节
    const observer = new IntersectionObserver(
      (entries) => {
        entries.forEach((entry) => {
          if (entry.isIntersecting) {
            const id = entry.target.id;
            links.forEach((link) => {
              link.classList.toggle(
                'toc__link--active',
                link.getAttribute('href') === '#' + id
              );
            });
          }
        });
      },
      { rootMargin: '-80px 0px -80% 0px' }
    );

    headings.forEach((h) => observer.observe(h));
  }

  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', tocInit);
  } else {
    tocInit();
  }
})();
```

```javascript
(function () {
  'use strict';

  function copyCodeInit() {
    document.querySelectorAll('pre').forEach((pre) => {
      const btn = document.createElement('button');
      btn.className = 'copy-btn';
      btn.textContent = '复制';
      btn.setAttribute('aria-label', '复制代码');

      btn.addEventListener('click', async () => {
        const code = pre.querySelector('code');
        if (!code) return;
        try {
          await navigator.clipboard.writeText(code.textContent);
          btn.textContent = '已复制!';
          setTimeout(() => { btn.textContent = '复制'; }, 2000);
        } catch {
          btn.textContent = '失败';
          setTimeout(() => { btn.textContent = '复制'; }, 2000);
        }
      });

      pre.appendChild(btn);
    });
  }

  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', copyCodeInit);
  } else {
    copyCodeInit();
  }
})();
```
